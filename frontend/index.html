🎵 Sistema de Recomendação de Músicas

Como usar

Este sistema aprende com as músicas curtidas por um usuário para sugerir novas músicas parecidas com seu perfil.

  1. Selecione um usuário.
  2. Curta algumas músicas para montar o histórico.
  3. Clique em Treinar Modelo para o sistema aprender o gosto do usuário.
  4. Depois clique em Gerar Recomendações para ver sugestões personalizadas.

🧠 Ações do Modelo

1. Treinar Modelo

Use esta ação depois que o usuário já tiver curtido algumas músicas.

O que isso faz: o sistema analisa o histórico de curtidas para aprender o padrão de preferência do usuário.

2. Gerar Recomendações

Recomendado usar esta ação após treinar o modelo.

O que isso faz: o sistema busca músicas parecidas com o perfil aprendido e mostra sugestões no painel principal.
